## Aditek Security Catalog

Aditek Security Catalog adalah aplikasi berbasis Next.js untuk menampilkan solusi security system dan infrastruktur enterprise.

## Menjalankan Proyek

Jalankan development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
Buka [http://localhost:3000](http://localhost:3000) untuk melihat aplikasi.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
Mulai ubah tampilan dari file `app/page.tsx`, lalu perubahan akan otomatis tampil.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
Proyek ini menggunakan [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) untuk optimasi font Geist.

## Fitur Utama

- Katalog solusi security dan infrastruktur untuk kebutuhan B2B.
- Halaman pencarian dengan filter domain solusi.
- Halaman detail solusi berisi kapabilitas teknis utama.
- Form autentikasi siap integrasi API.
